#N8N-Barbearia: Automação de Atendimento Inteligente

Tecnologias Utilizadas
n8n 
*Gemini API
* **Evolution API / Waha:** Integração com WhatsApp.
*Contabo/Inicialmente Docker

Como usar
1. Importe o arquivo `.json` presente neste repositório para o seu n8n.
2. Configure as credenciais das APIs de IA e WhatsApp.
3. Ative o workflow.
